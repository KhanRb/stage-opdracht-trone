## Frontend
De frontend is gebouwd met Vue 3 + TypeScript & Tailwind CSS.

**Installatie en starten:**

1. `cd frontend` <br>
2. `npm install` <br>
3. `npm run dev`

## Backend
De backend is gebouwd met Spring Boot + Java + JPA/Hibernate.

**Configuratie en starten:**

1. Kopieer `application.yml.example` naar `application.yml`
2. Vul de databaseconfiguratie in `application.yml` met de juiste credentials.
3. Run de main class `PoiApplication.java`, of start via Maven:

Voor de veiligheid worden de credentials niet in deze repository
gedeeld. De gegevens staan apart in de e-mail.
